# Few-shot Prompting

[Certain] Few-shot prompting means giving an AI model a small number of examples in the prompt so it can imitate the desired pattern.

Practical use cases:

- Rewriting in a specific style.
- Classification.
- Data extraction.
- Formatting.
- Tone matching.
