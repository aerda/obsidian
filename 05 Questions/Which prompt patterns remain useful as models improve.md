# Which prompt patterns remain useful as models improve?

Related source: [[Effective Prompting Fundamentals]]

[Probable] Some specific prompt techniques become less necessary as models improve, but core communication principles remain useful.

Patterns to monitor:

- Asking the model to think first.
- Providing examples.
- Breaking tasks into explicit steps.
- Defining roles.
- Asking for confidence levels.
