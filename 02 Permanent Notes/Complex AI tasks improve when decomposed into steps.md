# Complex AI tasks improve when decomposed into steps

[Probable] Complex AI tasks often improve when the user describes the process the AI should follow.

This is most useful when there are many valid ways to complete the task or when success depends on domain knowledge that the AI may not infer.

Example workflow:

1. Inspect the data.
2. Identify the best-performing products.
3. Compare the current quarter with the previous quarter.
4. Highlight unusual trends.
5. Suggest possible reasons for the trends.

Task decomposition turns a broad request into a guided workflow. It also makes it easier to identify where an AI answer went wrong.
