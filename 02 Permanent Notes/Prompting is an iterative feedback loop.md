# Prompting is an iterative feedback loop

[Certain] Effective prompting usually improves through iteration.

The first prompt does not need to be perfect. The user can refine the result by adding context, giving examples, changing constraints, asking for variations, requesting a different format, or starting a fresh conversation.

Iteration is especially important because AI systems change over time. A prompting technique that helps with one model may be less necessary or less effective with another.

Practical iteration loop:

1. Ask.
2. Evaluate the response.
3. Identify what failed.
4. Refine the prompt.
5. Try again or start a fresh conversation.
